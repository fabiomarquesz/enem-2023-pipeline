# ENEM 2023 Data Pipeline

Pipeline de engenharia de dados para os microdados do ENEM 2023 (INEP).

## Arquitetura
- **Bronze:** dados brutos ingeridos do INEP
- **Silver:** dados limpos e tipados
- **Gold:** agregações analíticas

## Stack
- Python 3.12
- PostgreSQL 16
- Apache Airflow 2.9
- Docker Compose
